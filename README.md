# Curso de Bootstrap 5 - Projeto Start

Este repositorio contem os arquivos iniciais do curso de Bootstrap 5.

Use esta pasta como ponto de partida para acompanhar as aulas. O aluno pode baixar ou clonar este repositorio no computador e abrir os arquivos HTML no navegador ou pelo Live Server do VS Code.

## Como usar

1. Baixe ou clone este repositorio.
2. Abra a pasta no VS Code.
3. Abra qualquer arquivo `.html` diretamente no navegador, ou use a extensao Live Server.
4. Acompanhe as aulas editando os arquivos conforme as explicacoes do curso.

## Estrutura

- `01-containers.html` ate `28-darkmode-toggle.html`: arquivos das aulas e exemplos.
- `template.html`: modelo base para novos exemplos.
- `assets/css/bootstrap.min.css`: Bootstrap 5 local.
- `assets/js/bootstrap.bundle.min.js`: JavaScript do Bootstrap 5 local.
- `assets/js/darkmodetoggle.js`: script auxiliar para alternancia de tema.

## Objetivo

Este e o repositorio start do curso. Ele foi preparado para que o aluno comece com a mesma base usada nas aulas.
